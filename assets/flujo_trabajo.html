FASE 3
Script 03
🔢 Índices Espectrales
Calcula NDVI (vegetación), NDWI (agua) y MNDWI (sedimentos/playas) sobre cada mosaico S2. Añade las 3 bandas al raster.
📥 sentinel2_YYYY | 📁 Output: 5 assets indices_YYYY
⬇
FASE 4
Script 04
🏖️ Detección de Arena
4 reglas para playa:
① MNDWI 0.0–0.35
② NDVI < 0.15
③ Rojo (B4) 0.08–0.35
④ SWIR (B11) < 0.12

Si cumple las 4 → 🏖️ arena/playa
📥 indices_YYYY + sentinel2_YYYY | 📁 bancos_arena_YYYY
Script 05
⛏️ Minería (Sentinel-2)
3 huellas de minería:
🟦 Piscinas: MNDWI 0.15–0.45 + NIR bajo
🟫 Dragas: NDVI <0.12 + SWIR alto
🟨 Agua turbia: MNDWI 0–0.25 + Rojo alto
📥 indices_YYYY + sentinel2_YYYY | 📁 mineria_YYYY
Script 05b
📡 Minería (Sentinel-1)
Radar ve a través de nubes:
🔩 Dragas: VV>-12 dB + ratio VV/VH>1.5
🟫 Suelo removido: VV>-15 dB + VH>-22 dB

Respaldo cuando S2 tiene nubes
📥 Sentinel-1 GRD | 📁 mineria_s1_YYYY
⬇

Script 06

📥 bancos_arena_2019,2025 + mineria_2019,2025 | 📁 cambio_arena + cambio_mineria
